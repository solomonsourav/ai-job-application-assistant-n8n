# AI Job Application Assistant using n8n

This is my first AI Agent project built using n8n.

The workflow helps job seekers generate professional job application support content such as follow-up emails, recruiter messages, resume keywords, and interview preparation notes.

## Features

- Generate job application follow-up emails
- Create LinkedIn recruiter messages
- Suggest resume keywords for job roles
- Prepare basic interview questions
- Improve job application messages
- Beginner-friendly AI automation workflow

## Tech Stack

- n8n
- AI Agent Node
- OpenAI Chat Model
- Manual Trigger

## Workflow Overview

Manual Trigger → AI Agent → Generated Response

## How to Use This Workflow

1. Download the workflow JSON file from this repository.
2. Open n8n.
3. Click **Import workflow**.
4. Upload the JSON file.
5. Add your OpenAI credentials.
6. Test the workflow using Manual Trigger.
7. Activate the workflow if required.
